## Columns

Notes:

##### Custom Classes 
|  Class | Function   |  
|--------|------------|
| N/A | Default columns Layout - text color black |  
| text-white | All text color white |
| hover-image-active | Show image when hover in column |
| stats-items | Show stats number with text |
